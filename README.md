## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18+)
- NPM or Yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/dataset-main.git
   cd dataset-main
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root and add your API keys:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_key
   VITE_FIREBASE_API_KEY=your_firebase_key
   ...
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) to see the magic.
